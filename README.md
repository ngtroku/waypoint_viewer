# waypoint_viewer
Waypointsを簡易的に可視化するツール

# Prerequisites
- python3
- matplotlib

# How to use
```
git clone https://github.com/ngtroku/waypoint_viewer.git
cd (cloned repository path)
python3 ./waypoint_viewer.py (waypoints file path)
```
## Example
```
python3 ./waypoint_viewer.py ./sample.csv
```
